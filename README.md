## 📋 FUNCTIONAL WORKFLOW
1.  **Identity Verification:** The system pings global intelligence nodes to verify the client's ISP brand and geo-location.
2.  **Base Latency Protocol:** Measures the "Unloaded" ping to verify the baseline speed of light through your fiber/WiFi link.
3.  **Stress Saturation:** The engine pulls 100MB binary chunks across 12 threads. This saturates the bandwidth, forcing the ISP to deliver its maximum advertised speed.
4.  **Neural Visualization:** Real-time data is piped into a 3nd-generation neural ring visualizer for millisecond-accurate speed tracking.
5.  **Uplink Projection:** Analyzes uplink node capacity based on saturation duration and packet loss metrics.
